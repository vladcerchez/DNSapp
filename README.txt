
---- Deschide-ti proiectul (IntelliJ) și rulați metoda main, folosiți comenzile < resolve
     "domain" ip > sau "ip" < domain >, puteți schimba dns serverul < use dns >
  

---- Open your project in IntelliJ and run the main method. Use the following commands within the program:

- To resolve a domain name to an IP address: <resolve 'domain' ip>
- To resolve an IP address to a domain name: <resolve 'ip' domain>
- You can change the DNS server by using the command: <use dns>"
